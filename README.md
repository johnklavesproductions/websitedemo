# Form & Field Studio — Website

The official website for Form & Field Studio, a boutique graphic design and brand identity studio.

## Live Site

Hosted via GitHub Pages at: `https://YOUR-USERNAME.github.io/YOUR-REPO-NAME`

## What's Included

- `index.html` — The full single-page website (HTML, CSS, and JS in one file)

## How to Update

To make changes to the site:
1. Edit `index.html` directly on GitHub (click the file → pencil icon) or in a code editor
2. Commit the changes to the `main` branch
3. GitHub Pages will automatically redeploy within a minute or two

## Tech Stack

- Pure HTML, CSS, and JavaScript — no frameworks or build tools needed
- Google Fonts: Bebas Neue, Playfair Display, DM Sans
